# 孙瑜｜产品经理个人作品站

这是一个用于秋招投递和面试前展示的个人作品站，重点呈现 C 端产品经理经历、AI 工具实践、vibe coding 能力和项目思考。

## 内容重点

- SHEIN C 端产品经理实习经历
- 多语言翻译补全与自动质检 Skill
- Claude Code / Codex / Dify / Python 工作流实践
- 好未来 AI Blog 与 Wiki 题库
- 百度 AgentBuilder 成长中心

## 本地运行

```bash
npm install
npm run dev -- --host 127.0.0.1
```

本地访问：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`，可部署到 Vercel、Netlify 或 GitHub Pages。
